# vim
編輯模式、一般模式、命令模式
>:e 檔案 : 在 vi 中開啟檔案

>:q! : 離開 vi (不儲存更動)

>:w : 儲存內容

>:wq : 儲存內容並離開

>i 或 o 或 a : 進入編輯模式

>[ESC鍵] : 從編輯模式、命令模式退回一般模式

>h : 左移

>j : 下移

>k : 上移

>l : 右移

>w : 移到下一個英文字

>yy : 複製游標所在該行

>p : 貼在下一行

>dd : 刪除游標所在該行


>u : 復原上一次更動

>. : 重複上一個動作

>gg : 移到第一行

>G : 移到最後一行

>:set nu : 顯示行號
